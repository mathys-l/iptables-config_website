Iptables-config

The Iptables-config script is an unofficial script for configure iptables on all you'r machine. With this script you can have you own configuration. 
Script has intuitive with question and adapt for starter.

Download

You can download the script with ssh, just copy and paste command below on you'r ssh.

sudo curl --tlsv1.3 https://raw.githubusercontent.com/mathys-l/iptables-config/master/iptables-config-launcher.sh | sudo bash
